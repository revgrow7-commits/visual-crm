# Visual CRM – Dashboard Gráfico de Vendas

Dashboard interativo para gestão de vendas, leads e propostas com integração Holdprint ERP.

## 🚀 Features

- **Dashboard**: KPIs, charts e pipeline visual
- **Leads, Clientes & Propostas**: Gestão completa com filtros
- **Relatórios**: 8 sub-módulos (Vendas, Pipeline, Conversão, Performance, Forecast, Deals em Risco, Produtos, Agendamentos)
- **Exportação**: CSV/Excel/PDF
- **Offline-ready**: Fallback para dados demo quando API indisponível
- **Responsivo**: Mobile-friendly interface

## 📱 Offline Mode

O app funciona em modo demo caso a API Holdprint não esteja disponível. Todos os dados de exemplo já estão embutidos.

## 🛠️ Tech Stack

- Vanilla HTML/JS/CSS (sem build)
- Chart.js 4.4
- SheetJS (xlsx.js)
- Holdprint ERP API

## 📊 Dados

Os dados são carregados em tempo real via API ou exibem dados de exemplo em modo demo.
